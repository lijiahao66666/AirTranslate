# AirTranslate 软件概述与所属领域

## 1. 软件名称
- 中文名称：灵译（AirTranslate）
- 英文名称：AirTranslate
- 软件类型：EPUB 电子书全本翻译工具（跨平台客户端 + 服务端 + 翻译 Worker）

## 2. 面向领域/行业
- 数字阅读行业（电子书阅读与内容处理）
- 跨语言出版与教育行业（多语种内容转换）
- AI 文本处理与翻译服务行业

## 3. 目标用户
- 有外文阅读需求的普通读者
- 需要批量处理 EPUB 内容的编辑/译者
- 需要将书籍内容快速转为目标语言的学习者

## 4. 软件定位
AirTranslate 面向“整本 EPUB 翻译”场景，强调：
1. 上传即翻译：从文件上传、任务处理到结果下载的一体化流程。
2. 多引擎策略：AI 高质量翻译与机器翻译免费通道并存。
3. 工程可部署：服务端、Worker、客户端解耦，适合本地/云端部署。
4. 本地优先体验：列表缓存与封面本地持久化，提升访问速度与稳定性。

## 5. 运行环境
- 服务端：Node.js（建议 16+）
- Worker：Python 3.11+
- AI 推理：vLLM（WSL2 + NVIDIA GPU）
- 客户端：Flutter（Web/Android/iOS）

## 6. 版本说明
- 当前材料基于 AirTranslate 当前代码版本（工作区目录：`AirTranslate/`）整理。
- 核心特征：任务“提交后待启动”、手动启动翻译流程、vLLM API 推理、客户端本地封面缓存。
